# 哈耶克的文明：市场经济 v1.50版本《威斯特伐利亚》

本次更新（代号：*威斯特伐利亚*）将游戏重点从单一的国家建设扩展到了复杂的地缘政治博弈。我们引入了基于时代的条约体系、完整的附庸宗主机制、海外资本输出以及深度的官员忠诚度模拟。

## 🌏 核心重构：外交系统 2.0 (Diplomacy 2.0)

外交不再仅仅是宣战与结盟，现在它是你延伸经济触手、控制世界资源的工具。

### 1. 深度条约体系 (Deep Treaty System)
所有的外交条约现在都受到**时代进程 (Epochs)** 的限制，随着文明发展解锁更高级的互动：

| 条约类型 | 解锁时代 | 核心效果 |
| :--- | :--- | :--- |
| **互不侵犯条约** | 古典时代 | 战争风险大幅降低，商人派驻安全性 +100%。 |
| **贸易协定** | 古典时代 | 增加 3 个额外商路槽位，双方关税降低 20%~25%。 |
| **自由贸易协定** (FTA) | 探索时代 | **零关税**，且激活**市场价格联动**（两国物价趋同）。 |
| **投资协议** | 探索时代 | 允许在对方领土进行建设，利润受保护，国有化风险减半。 |

> **新机制：动态外交成本**
> 外交行动的成本（白银/行政点）现在会随你的国库规模动态调整。再也没有大后期“花零钱买和平”的好事了。

### 2. 统一附庸政策体系 (Unified Vassalage & Policy System)
所有附庸国共享统一的基础地位，但你可以通过**深度定制政策**来决定你们的关系实质：

*   **� 劳工政策**: 自主决定是维持“正常雇佣”，还是进行“压榨剥削”甚至“强制劳动”。
*   **⚖️ 贸易政策**:
    *   **自由贸易**: 允许附庸自主贸易，换取低独立倾向。
    *   **倾销市场**: 强制附庸购买你的过剩商品。
    *   **资源掠夺**: 以极低价格强行收购附庸原材料。
*   **🏛️ 治理权限**: 从“完全自治”到“总督直辖”的平滑过渡。
*   **👮 控制手段**: 现在你可以组合使用多种手段来压制独立倾向：
    *   **派遣总督**: 利用官员能力进行精细化管理（效果受官员属性影响）。
    *   **驻军占领**: 简单粗暴的武力镇压，但会激起民愤。
    *   **文化同化**: 长期降低由于民族主义带来的独立倾向。

> **设计理念**：不再有预设的“傀儡国”，你自己通过政策组合把它变成傀儡。

### 3. 全球资本化：海外投资 (Overseas Investment)
不仅可以由内向外投资，AI 资本也会进入你的市场（双向交互）：

*   **资本输出**：在附庸国或条约国建设农场、矿山、工厂。
    *   **本地模式**：利用当地廉价劳动力，赚取汇差。
    *   **倾销模式**：向当地倾销过剩产品，打击当地工业。
    *   **回购模式**：掠夺当地稀缺资源（如石油、铁矿）。
*   **外资引入**：盟友 AI 会根据市场价格差，主动在你国内投资建设。你可以对它们收税，甚至在战争时将其**国有化（没收）**。

### 4. 国际组织与联盟
*   **经济共同体**: 多国组成的贸易联盟，成员间享有额外的贸易效率加成。
*   **多轮谈判**: 外交不再是一锤子买卖，现在支持多次讨价还价。

## 🏛️ 治理重构：官员系统 2.0 (Officials Overhaul)

告别千篇一律的工具人，每位官员都是鲜活的政治个体。

### 1. 动态能力生成
官员的效果完全由其四维属性（行政/军事/外交/威望）决定，且受**强度因子**修正：
*   **高行政** → 极高的税收效率、建筑成本减免。
*   **高军事** → 战时生产力暴涨、军队维护费减半。
*   **高外交** → 贸易利润加成、改善邻国关系。
*   **新机制**：属性不再只是摆设，它直接决定了 Buff 的数值强度（0.3x ~ 1.5x）。

### 2. 出身决定立场 (Stratum Background)
官员的**出身阶层**决定了他们的政治基本盘：
*   **地主出身**：死保农业利益，反对工业化征地。
*   **资本家出身**：狂热追求自由贸易，反对高税收。
*   **自耕农出身**：关注粮食安全与人口增长，厌恶战争。

### 3. 忠诚度与政变 (Loyalty & Coup)
*   **忠诚度 (0-100)**：受政治诉求满足度、薪资发放及时性、国家稳定度影响。
*   **政变危机**：忠诚度 < 25 的强力官员可能发动政变，直接推翻政府或引发内战。
*   **生杀大权**：你可以解雇、流放甚至处决不听话的权臣——每种选择都有代价。
