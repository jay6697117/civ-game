## 🎯 核心概念

### 兵种分类
- **步兵** 🗡️ - 防御强，克制骑兵
- **弓箭手** 🏹 - 远程攻击，克制步兵
- **骑兵** 🐎 - 高机动，克制弓箭手
- **攻城** 💣 - 重火力，范围伤害

### 克制链条
```
长矛兵 → 骑兵 → 弓箭手 → 步兵 → 长矛兵
```

## 💰 资源管理

### 招募限制
- **行政力**: 总消耗 ≤ 行政容量
- **人口**: 军队人口 ≤ 总人口 × 30%
- **资源**: 足够的招募成本

### 维护成本控制
- 维护成本 < 黄金收入 × 70%
- 和平时期减少军队规模
- 优先保留高级单位

## ⚔️ 战斗技巧

### 兵种配比

**平衡配置**:
- 40% 步兵
- 30% 弓箭手
- 20% 骑兵
- 10% 攻城

**快速突击**:
- 70% 骑兵
- 30% 龙骑兵

**坚固防御**:
- 50% 重装步兵
- 30% 弩兵
- 20% 长矛兵

### 克制运用

| 敌军主力 | 我方对策 |
|---------|---------|
| 骑兵多 | 招募长矛兵 |
| 步兵多 | 招募弓箭手 |
| 弓箭手多 | 招募骑兵 |
| 混合部队 | 平衡配置 |

### 战斗力计算

```
基础战斗力 = Σ[(攻击+防御) × 数量] × (1 + 时代差×0.1)
最终战斗力 = 基础战斗力 × 克制加成 × (1 + 军事buff) × 随机(0.85-1.15)
```

### 胜负判定
- 战斗力占比 > 50% = 胜利
- 战斗力优势 > 80% = 压倒性胜利

## ⚠️ 常见错误

### ❌ 避免
1. 单一兵种（易被克制）
2. 过度扩军（维护成本过高）
3. 忽视克制关系
4. 行政力不足时强行招募
5. 低时代单位对抗高时代

### ✅ 推荐
1. 兵种多样化
2. 控制军队规模
3. 针对性招募
4. 优先升级时代
5. 满足军人需求获得buff

## 💡 专家技巧

### 1. 时代压制
- 每高一个时代 = +10%战斗力
- 优先升级时代获得优势

### 2. 军事buff
- 满足军人需求 = +20%军事力量
- 满足骑士需求 = +25%军事力量
- 总buff可叠加

### 3. 战利品最大化
- 压倒性胜利 = 30%掠夺率
- 普通胜利 = 15%掠夺率
- 提升战斗力差距

### 4. 损失最小化
- 克制关系正确 = 减少损失
- 压倒性胜利 = 损失减半
- 避免劣势战斗

## 📋 操作流程

### 招募单位
1. 打开**军事**标签页
2. 查看可招募单位
3. 检查资源和限制
4. 点击**招募**按钮
5. 等待训练完成

### 发起攻击
1. 打开**外交**标签页
2. 选择目标国家
3. 点击**进攻**按钮
4. 查看战斗结果
5. 获得战利品

### 管理军队
1. 查看军队概览
2. 检查维护成本
3. 解散不需要的单位
4. 保持行政力平衡
