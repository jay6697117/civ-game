## 🔧 第三步：基本操作

### 操作1: 选择国家

```javascript
// 游戏开始时选择国家
const myNation = COUNTRIES.find(c => c.id === 'silk_empire');

// 查看国家特性
console.log(myNation.specialAbilities);
// 输出: [
//   { type: 'culture_influence', value: 0.3, desc: "文化影响力 +30%" },
//   { type: 'admin_efficiency', value: 0.2, desc: "行政效率 +20%" }
// ]
```

### 操作2: 激活政令

```javascript
// 查看可用政令
const availableDecrees = DECREES.filter(d => d.unlockEpoch <= currentEpoch);

// 激活政令
const decree = DECREES.find(d => d.id === 'free_trade');
if (canAfford(decree.cost)) {
  decree.active = true;
  applyDecreeEffects(decree);
}
```

### 操作3: 发展产业链

```javascript
// 查看产业链状态
const chain = INDUSTRY_CHAINS.luxury_chain;
const efficiency = calculateChainEfficiency('luxury_chain');

// 升级产业链
upgradeChain('luxury_chain', 'trade_network');
```

## 💡 第四步：理解核心机制

### 机制1: 政令协同

**当你同时激活这些政令时**:
- 自由贸易
- 殖民特许
- 市场监管

**会触发"贸易帝国"组合**:
- 额外获得: 贸易收益 +30%, 银币 +5/日, 外交 +15
- 这比单独激活效果更强！

### 机制2: 产业链效率

**完整产业链** (所有阶段都有建筑):
- 基础效率: 100%
- 完整加成: +20%
- 总效率: 120%

**不完整产业链** (缺少某个阶段):
- 基础效率: 100%
- 瓶颈惩罚: -30%
- 总效率: 70%

**建议**: 确保产业链各阶段都有建筑！

### 机制3: 阶层反馈

**阶层好感度影响政令效果**:

高好感度 (>70%):
```
商人好感 75% → 贸易政令效果 +15%
```

低好感度 (<30%):
```
商人好感 25% → 贸易政令效果 -25%
```

**建议**: 保持相关阶层的高好感度！

## ⚠️ 常见错误

### 错误1: 政令冲突

❌ **错误做法**:
```
同时激活: 自由贸易 + 国家专营
结果: 效率 -30%, 稳定度 -15 (政令冲突!)
```

✅ **正确做法**:
```
只激活其中一个，根据你的策略选择
```

### 错误2: 忽视产业链瓶颈

❌ **错误做法**:
```
只建采矿建筑，不建加工建筑
结果: 矿石堆积，无法转化为有用资源
```

✅ **正确做法**:
```
确保产业链各阶段都有建筑
采矿 → 冶炼 → 制造 → 应用
```

### 错误3: 忽视阶层好感度

❌ **错误做法**:
```
激活贸易政令，但商人好感度只有20%
结果: 政令效果大打折扣
```

✅ **正确做法**:
```
先提升商人好感度到70%以上
然后激活贸易政令，效果更好
```

## 💬 常见问题

### Q: 哪个国家最强？
A: 没有最强的国家，只有最适合你玩法风格的国家。贸易流选沙漠商队，军事流选草原汗国，文化流选丝绸帝国。

### Q: 政令可以随时激活和取消吗？
A: 可以，但取消政令不会退还行政成本。建议谨慎选择。

### Q: 产业链必须全部发展吗？
A: 不必须，但建议至少发展2-3条产业链以保持平衡。专精一条也可以，但风险较大。

### Q: 如何提升阶层好感度？
A: 满足阶层的资源需求，激活有利于该阶层的政令，发展相关产业链。

### Q: 系统联动太复杂了怎么办？
A: 先从简单的开始，逐步学习。使用调试工具查看效果来源，帮助理解。
