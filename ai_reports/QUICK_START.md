# 🚀 时代升级系统 - 快速开始指南

## 立即体验

### 1. 启动游戏
```bash
cd civ-game
npm run dev
```

### 2. 快速升级到青铜时代

#### 第一步：积累科研 (目标: 500)
- 建造 **图书馆** (成本: 木材200, 石料50)
- 图书馆产出: 科研 +2/秒
- 需要时间: 约4-5分钟

#### 第二步：增加人口 (目标: 20)
- 建造 **简陋小屋** (成本: 木材20, 粮食20)
- 每个小屋增加人口上限 +3
- 建造 6-7 个小屋
- 保持粮食充足让人口自然增长

#### 第三步：积累升级资源
- 粮食: 500 (建造农田)
- 木材: 300 (建造伐木场)
- 石料: 100 (建造采石场)
- 黄金: 50 (通过税收或贸易获得)

#### 第四步：执行升级
1. 点击"科技"标签页
2. 查看顶部的"时代升级"区域
3. 确认所有条件为绿色✅
4. 点击闪烁的"升级到青铜时代"按钮
5. 🎉 恭喜！你已进入青铜时代！

## 升级后的变化

### 视觉变化
- 背景色从灰色变为橙色调
- 地图方块颜色改变
- 界面整体氛围更温暖

### 游戏性变化
- 采集效率提升 15%
- 解锁新建筑：庄园、木屋、兵营、教堂等
- 解锁新科技：封建制度、神学
- 军事力量提升 10%

### 数值变化示例
**升级前:**
- 农田产出: 4 粮食/秒

**升级后:**
- 农田产出: 4.6 粮食/秒 (提升15%)

## 💡 新手建议

### 建筑建造顺序
1. **农田** x2 - 保证粮食供应
2. **伐木场** x2 - 获取木材
3. **简陋小屋** x3 - 增加人口
4. **图书馆** x1 - 开始科研
5. **采石场** x1 - 获取石料
6. **市场** x1 - 产出黄金

### 常见错误
❌ **过早升级** - 资源不足导致后续发展困难
❌ **忽视人口** - 没有足够劳动力
❌ **单一发展** - 只关注科研忽视其他资源
❌ **不看加成** - 不了解升级带来的实际收益

### 最佳实践
✅ **平衡发展** - 同时关注科研、人口、资源
✅ **提前规划** - 查看下一时代的要求
✅ **资源储备** - 保持一定的资源余量
✅ **利用加成** - 升级后扩大生产规模

## 🎯 进阶技巧

### 快速升级策略
1. **科研优先流**
   - 早期多建图书馆
   - 快速达到科研要求
   - 适合追求快速进步

2. **稳健发展流**
   - 平衡各项资源
   - 确保升级后有足够资源继续发展
   - 适合新手

3. **经济优先流**
   - 先发展经济建筑
   - 积累大量资源后一次性升级
   - 适合长期规划

### 时代升级时机
**最佳升级时机:**
- 当前时代建筑已充分发展
- 资源产出稳定且有盈余
- 人口接近上限
- 科研和文化持续增长

**不宜升级时机:**
- 资源刚好够升级成本
- 人口不足导致建筑缺乏劳动力
- 行政压力过大
- 正在进行外交活动

## 📊 各时代升级时间参考

| 时代 | 预计游戏时间 | 主要挑战 |
|------|------------|---------|
| 石器→青铜 | 10-15分钟 | 科研积累 |
| 青铜→封建 | 20-30分钟 | 文化发展 |
| 封建→工业 | 40-60分钟 | 产业链建设 |

*注：时间会根据游戏速度和策略有所不同*

## 🔍 故障排查

### 问题：升级按钮是灰色的
**检查清单:**
- [ ] 科研是否达标？
- [ ] 人口是否达标？
- [ ] 文化是否达标？（封建时代起）
- [ ] 所有升级资源是否充足？

### 问题：升级后感觉没变化
**解决方案:**
- 查看资源产出速率（右侧数字）
- 建造更多生产建筑来体现加成
- 查看日志确认升级成功

### 问题：资源不够升级
**建议:**
- 建造对应的资源生产建筑
- 通过外交贸易获取资源
- 使用手动采集补充资源
- 调整游戏速度加快积累
