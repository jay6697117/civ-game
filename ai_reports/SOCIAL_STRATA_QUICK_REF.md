# 社会阶层系统 - 快速参考

## 🎯 核心概念

### 好感度 (Approval)
- **满意 (≥70)**: 提供正面Buff ✅
- **中立 (40-69)**: 无特殊效果 ○
- **不满 (<40)**: 产生负面Debuff ❌

### 影响力 (Influence)
阶层对国家决策的影响程度，由人口和财富决定。

### 财富 (Wealth)
阶层掌握的经济资源，影响其影响力。

## 🎮 快速策略

### 早期 (石器-青铜)
1. ✅ 保证粮食充足（自耕农基础）
2. ✅ 控制奴隶数量
3. ✅ 建立木材-石料产业链

### 中期 (封建)
1. ✅ 发展文化（教堂）
2. ✅ 建立黄金收入（市场）
3. ✅ 平衡地主需求

### 后期 (工业-信息)
1. ✅ 供应工具给工人
2. ✅ 满足资本家需求
3. ✅ 全面资源供应

## ⚠️ 危机应对

### 好感度 < 40 时
1. 🔍 检查资源需求
2. 🏭 优先生产短缺资源
3. 📜 调整政令
4. 📊 监控稳定度

### 关键阶层不满
- **军人**: 可能哗变！立即供应黄金
- **地主**: 税收暴跌！提供黄金和文化
- **资本家**: 工业崩溃！确保黄金、工具、科研
- **官员**: 行政瘫痪！提供黄金和文化

## 💡 专家技巧

### 资源优先级
1. **粮食** - 所有阶层基础
2. **黄金** - 精英阶层关键
3. **文化** - 稳定度保障
4. **工具** - 工业时代核心

### 最优配置目标
- 所有阶层好感度 > 70
- 稳定度 > 80
- 多个正面Buff生效
- 零负面Debuff

### 影响力管理
- 高影响力阶层（资本家、地主、官员）优先满足
- 低影响力阶层（奴隶、佃农）可适当忽略
- 平衡精英和平民的需求

## 📈 数值参考

### 好感度影响因素
- 资源满足 > 200%: +10
- 资源满足 100-200%: 0
- 资源满足 50-100%: -10
- 资源满足 < 50%: -20

### 影响力计算
```
影响力 = (基础影响力 × 人口) + (财富因子 × 2)
```

### 财富计算
```
财富 = 人口 × 财富权重
```

## 🎯 成就挑战

- ⭐ **精英统治**: 地主+资本家影响力 > 50%
- ⭐ **人民力量**: 自耕农+工人人口 > 70%
- ⭐ **完美平衡**: 稳定度 = 100
