# 游戏系统扩展设计文档

## 概述

本文档详细说明了为文明游戏新增的四大系统扩展：国家系统、政令系统、产业链系统和系统联动机制。这些扩展旨在增强游戏的策略深度、可玩性和沉浸感。

---

## 一、国家系统扩展

### 1.1 设计理念

在原有8个国家的基础上，新增12个具有独特文化背景和历史特色的国家，每个国家都有：
- 独特的文化特质
- 特殊的经济偏好
- 专属的能力加成
- 不同的外交风格

### 1.2 新增国家列表

#### 1.2.1 草原汗国 (Steppe Horde)
- **政体类型**: 游牧部落
- **时代**: 古典时代 - 封建时代
- **文化特质**: 
  - 军事导向，骑兵文化
  - 掠夺经济模式
  - 机动性强
- **特殊能力**:
  - 掠夺收益 +30%
  - 骑兵训练成本 -20%
- **经济偏好**: 粮食、铁矿、工具
- **外交风格**: 侵略性强 (0.75)

#### 1.2.2 沙漠商队 (Desert Caravan)
- **政体类型**: 商业联盟
- **时代**: 古典时代 - 启蒙时代
- **文化特质**:
  - 贸易专精
  - 市场稳定性高
  - 外交能力强
- **特殊能力**:
  - 贸易收益 +25%
  - 市场波动 -15%
- **经济偏好**: 香料、银币、文化
- **外交风格**: 和平贸易 (0.25)

#### 1.2.3 雨林帝国 (Jungle Empire)
- **政体类型**: 神权王朝
- **时代**: 青铜时代 - 探索时代

#### 1.2.4 山地氏族 (Mountain Clans)

  - 采矿产出 +25%
  - 防御力 +30%
- **经济偏好**: 石料、铁矿、铜矿

#### 1.2.5 河谷邦联 (River Confederation)

- **经济偏好**: 粮食、纸草、砖块、文化
- **外交风格**: 和平 (0.3)

#### 1.2.6 群岛海权 (Island Thalassocracy)

#### 历史渊源
- **草原汗国** vs **丝绸帝国**: 历史冲突，贸易竞争
- **沙漠商队** ↔ **自由商会**: 贸易伙伴，经济联盟
- **雨林帝国** ↔ **圣教骑士团**: 宗教对立
- **山地氏族** ↔ **河谷邦联**: 资源互补，和平共处
- **群岛海权** ↔ **北境公国**: 贸易竞争
- **革命共和国** vs **所有君主制国家**: 意识形态对立

#### 外交修正
- 相同政体类型: +10 外交关系
- 对立政体类型: -15 外交关系
- 贸易伙伴: +20 外交关系
- 历史冲突: -25 外交关系

## 二、政令系统深化

### 2.1 设计理念

在原有8个政令基础上，新增30个政令，覆盖经济、军事、文化、社会、科技五大类别，并建立政令间的协同和冲突机制。

### 2.2 政令分类

#### 2.2.1 经济政令 (8个)
1. **市场监管** - 稳定物价，保护商人
2. **自由贸易** - 提升贸易收益，增加波动
3. **国家专营** - 政府垄断，增加财政
4. **行会特许** - 提升工匠产出
5. **贸易保护** - 保护本国产业
6. **殖民特许** - 海外贸易加成
7. **基建计划** - 大规模基础设施投资
8. **徭役筑路** - 征调劳力修路

#### 2.2.2 军事政令 (5个)
1. **征兵法** - 快速扩军，损害经济
2. **职业军队** - 精锐部队，成本高
3. **军事学院** - 提升军队素质
4. **战时经济** - 军事生产优先
5. **军官团制** - 贵族军官，提升指挥

#### 2.2.3 文化政令 (6个)
1. **公共教育** - 提升民众素质
2. **大学特许** - 推动学术发展
3. **国教制度** - 宗教凝聚力
4. **世俗国家** - 政教分离
5. **宗教宽容** - 多元信仰
6. **文化赞助** - 繁荣艺术

#### 2.2.4 社会政令 (7个)
1. **土地改革** - 重新分配土地
2. **封建特权** - 维护贵族权力
3. **平等法令** - 废除阶级特权
4. **学徒制度** - 技术传承
5. **福利制度** - 社会保障
6. **贵族统治** - 强化贵族地位
7. **强制劳动** - 超时工作

#### 2.2.5 科技政令 (4个)
1. **科研资助** - 加速技术进步
2. **专利法** - 保护发明创造
3. **出版自由** - 促进知识传播
4. **工人公约** - 改善工人待遇

### 2.3 政令协同效果

#### 协同组合示例

**贸易帝国组合**
- 政令: 自由贸易 + 殖民特许 + 市场监管
- 效果: 贸易收益 +30%, 每日 +5 银币, 外交 +15
- 描述: 完整的贸易政策体系带来巨大收益

**学术繁荣组合**
- 政令: 公共教育 + 大学特许 + 科研资助
- 效果: 科研 +50%, 文化 +30%, 学者好感 +30
- 描述: 全面的教育和科研政策推动文明进步

**军国主义组合**
- 政令: 征兵法 + 军事学院 + 战时经济
- 效果: 军事力量 +60%, 征兵速度 +50%, 军队上限 +40%
- 惩罚: 生产 -15%, 稳定度 -15
- 描述: 全面军事化带来强大战力但损害经济

**社会革命组合**
- 政令: 土地改革 + 平等法令 + 福利制度
- 效果: 下层阶级好感 +40, 稳定度 +20, 人口增长 +30%
- 惩罚: 上层阶级好感 -50, 每日 -8 银币
- 描述: 彻底的社会改革重塑阶级结构

**工匠传统组合**
- 政令: 行会特许 + 学徒制度 + 专利法
- 效果: 手工业产出 +35%, 工匠好感 +35, 质量 +25%
- 描述: 完善的工匠培养体系提升制造业

**基建强国组合**
- 政令: 基建计划 + 徭役筑路 + 市场监管
- 效果: 生产 +25%, 采集效率 +20%, 整体效率 +20%
- 描述: 完善的基础设施提升整体效率

### 2.4 政令冲突机制

#### 冲突组合示例

**政策矛盾**
- 政令: 自由贸易 + 国家专营
- 惩罚: 效率 -30%, 稳定度 -15
- 描述: 自由贸易与国家垄断相互抵消

**阶级冲突**
- 政令: 平等法令 + 封建特权
- 惩罚: 稳定度 -30, 生产 -20%
- 描述: 平等与特权政策引发严重冲突

**信仰分裂**
- 政令: 国教制度 + 世俗国家
- 惩罚: 稳定度 -25, 文化 -20%
- 描述: 宗教与世俗政策相互对立

### 2.5 政令长期影响

每个政令都有短期效果和长期影响：

**短期效果** (立即生效)
- 资源产出变化
- 阶层好感度变化
- 经济指标调整

**长期影响** (累积效果)
- 社会结构演变
- 文化氛围转变
- 技术发展方向
- 外交关系变化

## 三、产业链系统

### 3.1 设计理念

建立完整的产业上下游关系网络，从原材料采集到最终产品消费，形成闭环经济系统。

### 3.2 产业链类型

#### 3.2.1 粮食产业链
```
农业生产 → 粮食储存 → 民众消费
├─ 建筑: 农田、庄园
├─ 工人: 自耕农、佃农
└─ 升级路径:
   ├─ 灌溉系统 (青铜时代): 效率 +15%, 产出 +10%
   ├─ 轮作制度 (古典时代): 效率 +20%, 可持续性 +15%
   └─ 机械化农业 (工业时代): 效率 +40%, 劳动力 -30%
```

#### 3.2.2 木材产业链
```
森林采伐 → 木材加工 → 建筑与制造
├─ 建筑: 伐木场 → 锯木厂
├─ 工人: 樵夫 → 工人、工匠
├─ 产品: 木材 → 木板
└─ 升级路径:
   ├─ 林业管理 (古典时代): 效率 +15%, 可持续性 +20%
   └─ 先进锯木厂 (探索时代): 加工效率 +25%, 浪费 -15%
```

#### 3.2.3 采矿产业链
```
矿石开采 → 初级冶炼 → 精密制造 → 工业应用
├─ 建筑: 采石场、铜矿、铁矿、煤矿 → 熔炉、锻造厂 → 工厂、钢铁厂
├─ 工人: 矿工 → 工匠、工程师 → 工程师、工人
├─ 产品: 石料、铜、铁、煤 → 工具、钢材
└─ 升级路径:
   ├─ 深井采矿 (封建时代): 开采量 +25%, 深度 +2
   ├─ 高炉技术 (启蒙时代): 加工效率 +35%, 整体效率 +20%
   └─ 工业综合体 (工业时代): 效率 +50%, 产出 +30%
```

#### 3.2.4 知识产业链
```
纸张生产 → 知识记录 → 知识传播 → 社会应用
├─ 建筑: 纸草田 → 图书馆、抄写室 → 印刷所、大学
├─ 工人: 自耕农、抄写员 → 抄写员、学者 → 学者、抄写员
├─ 产品: 纸草 → 文化 → 文化、科研
└─ 升级路径:
   ├─ 印刷机 (启蒙时代): 效率 +40%, 传播 +50%
   └─ 公共图书馆 (探索时代): 可及性 +30%, 文化 +20%
```

#### 3.2.5 奢侈品产业链
```
原料采集 → 精加工 → 上层消费
├─ 建筑: 香料市场、咖啡种植园 → 咖啡馆、贸易港
├─ 工人: 商人、佃农 → 商人、工匠
├─ 产品: 香料、咖啡 → 文化
└─ 升级路径:
   ├─ 贸易网络 (启蒙时代): 效率 +30%, 价格 +20%
   └─ 奢侈品垄断 (启蒙时代): 利润 +50%, 影响力 +25%
```

#### 3.2.6 军事产业链
```
原料供应 → 武器制造 → 军队训练 → 军事行动
├─ 建筑: 铁矿、铜矿 → 锻造厂、军械库 → 兵营、训练场
├─ 工人: 矿工 → 工匠、工程师 → 士兵
├─ 产品: 铁、铜 → 武器 → 军事力量
└─ 升级路径:
   ├─ 标准化武器 (探索时代): 效率 +25%, 成本 -15%
   ├─ 军事后勤 (启蒙时代): 补给 +30%, 机动性 +20%
   └─ 军工厂 (工业时代): 生产 +50%, 质量 +30%
```

### 3.3 产业链效率系统

#### 3.3.1 完整产业链加成
当产业链各环节齐全时:
- 效率 +20%
- 产出 +15%

#### 3.3.2 专业化加成
专注发展某一产业链时:
- 效率 +25%
- 质量 +20%

#### 3.3.3 规模经济
同类建筑数量达到阈值时:
- 5个: 成本 -10%
- 10个: 成本 -20%, 效率 +10%
- 20个: 成本 -30%, 效率 +20%

#### 3.3.4 技术外溢
高级产业带动相关产业:
- 相关产业效率 +15%

### 3.4 产业链瓶颈

#### 资源短缺
- 效率 -30%
- 产出 -25%

#### 加工瓶颈
- 效率 -25%
- 浪费 +15%

#### 劳动力短缺
- 效率 -40%
- 产出 -30%

#### 基础设施限制
- 效率 -20%
- 成本 +15%

### 3.5 产业链发展路径

每个产业链都有多条发展路径可选择:

**粮食产业链**
- 集约化农业: 提高单位产量
- 粗放式农业: 扩大种植面积

**采矿产业链**
- 重工业化: 大规模工业生产
- 精密制造: 高质量产品

**知识产业链**
- 大众教育: 普及教育
- 精英研究: 培养顶尖人才

## 四、系统联动机制

### 4.1 国家与政令联动

不同国家对政令有不同效果:

#### 草原汗国
- **增强政令**: 征兵法 (征兵速度 +20%, 成本 -25%)
- **限制政令**: 封建特权 (效率 -30%)

#### 沙漠商队
- **增强政令**: 自由贸易 (贸易收益 +15%, 市场稳定 +10%)
- **限制政令**: 国家专营 (商人好感 -20, 效率 -25%)

#### 丝绸帝国
- **增强政令**: 贵族统治 (行政容量 +10%, 效率 +15%)
- **增强政令**: 行会特许 (手工业产出 +10%)

#### 革命共和国
- **增强政令**: 平等法令 (生产 +10%, 稳定度 +10)
- **限制政令**: 封建特权 (稳定度 -25, 全阶层好感 -15)

### 4.2 阶层与政令反馈

阶层好感度影响政令效果:

#### 高好感度 (>70%)
- 生产 +10%
- 效率 +15%
- 忠诚度 +20%

#### 低好感度 (<30%)
- 生产 -15%
- 效率 -20%
- 抵抗 +25%

#### 特定阶层影响
- **商人**: 影响贸易政令效果 (±15% 贸易收益)
- **工人**: 影响工业政令效果 (±25% 工业产出)
- **农民**: 影响农业政令效果 (±20% 采集效率)
- **贵族**: 影响行政政令效果 (±15 稳定度)
- **学者**: 影响科研政令效果 (±25% 科研效率)

### 4.3 产业链与阶层互动

产业链发展影响阶层:

#### 粮食产业链
- **主要阶层**: 自耕农、佃农
- **发展效果**:

## 五、游戏平衡性设计

### 5.1 国家平衡

每个国家都有优势和劣势:
- 军事强国: 经济或文化较弱
- 贸易强国: 军事力量有限
- 文化强国: 扩张能力受限

### 5.2 政令平衡

每个政令都有代价:
- 强力加成必有显著惩罚
- 协同组合需要高行政成本
- 冲突组合会严重削弱效果

### 5.3 产业链平衡

产业链发展需要平衡:
- 专精某一产业链会忽视其他
- 完整产业链需要大量资源投入
- 产业链瓶颈会严重影响效率

### 5.4 阶层平衡

阶层关系需要平衡:
- 满足上层需求会忽视下层
- 社会改革会引发动荡
- 阶层冲突会降低稳定度

## 六、实施建议

### 6.1 分阶段实施

**第一阶段**: 国家系统扩展
- 添加新国家配置
- 实现国家特性效果
- 测试平衡性

**第二阶段**: 政令系统深化
- 添加新政令
- 实现协同/冲突机制
- 测试政令组合效果

**第三阶段**: 产业链系统
- 实现产业链逻辑
- 添加升级路径
- 测试经济平衡

**第四阶段**: 系统联动
- 实现各系统间联动
- 添加动态平衡机制
- 测试整体平衡

### 6.2 测试重点

1. **数值平衡**: 确保没有过强或过弱的选项
2. **策略深度**: 验证多种策略的可行性
3. **玩家体验**: 确保系统易于理解和使用
4. **性能优化**: 确保复杂计算不影响游戏流畅度

### 6.3 后续扩展

- 更多国家和文明
- 更复杂的外交系统
- 动态事件系统
- 多人对战模式

- **12个新国家**, 每个都有独特特性
