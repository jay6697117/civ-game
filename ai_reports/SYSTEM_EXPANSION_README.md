## 📜 二、政令系统深化

### 政令数量
- **原有**: 8个政令
- **新增**: 30个政令
- **总计**: 38个政令

### 政令分类

#### 经济政令 (8个)
- 市场监管、自由贸易、国家专营
- 行会特许、贸易保护、殖民特许
- 基建计划、徭役筑路

#### 军事政令 (5个)
- 征兵法、职业军队、军事学院
- 战时经济、军官团制

#### 文化政令 (6个)
- 公共教育、大学特许、国教制度
- 世俗国家、宗教宽容、文化赞助

#### 社会政令 (7个)
- 土地改革、封建特权、平等法令
- 学徒制度、福利制度、贵族统治、强制劳动

#### 科技政令 (4个)
- 科研资助、专利法、出版自由、工人公约

### 政令机制

#### 协同组合 (6个)
1. **贸易帝国**: 自由贸易 + 殖民特许 + 市场监管
   - 贸易收益 +30%, 银币 +5/日, 外交 +15

2. **学术繁荣**: 公共教育 + 大学特许 + 科研资助
   - 科研 +50%, 文化 +30%, 学者好感 +30

3. **军国主义**: 征兵法 + 军事学院 + 战时经济
   - 军事力量 +60%, 征兵 +50%, 但生产 -15%

4. **社会革命**: 土地改革 + 平等法令 + 福利制度
   - 下层好感 +40, 人口增长 +30%, 但上层好感 -50

5. **工匠传统**: 行会特许 + 学徒制度 + 专利法
   - 手工业 +35%, 工匠好感 +35, 质量 +25%

6. **基建强国**: 基建计划 + 徭役筑路 + 市场监管
   - 生产 +25%, 采集 +20%, 效率 +20%

#### 冲突组合 (3个)
- 自由贸易 ⚔️ 国家专营
- 平等法令 ⚔️ 封建特权
- 国教制度 ⚔️ 世俗国家

## 🏭 三、产业链系统

### 产业链类型 (6条)

#### 1. 粮食产业链
```
农业生产 → 粮食储存 → 民众消费
```
- 升级: 灌溉系统、轮作制度、机械化农业

#### 2. 木材产业链
```
森林采伐 → 木材加工 → 建筑制造
```
- 升级: 林业管理、先进锯木厂

#### 3. 采矿产业链
```
矿石开采 → 初级冶炼 → 精密制造 → 工业应用
```
- 升级: 深井采矿、高炉技术、工业综合体

#### 4. 知识产业链
```
纸张生产 → 知识记录 → 知识传播 → 社会应用
```
- 升级: 印刷机、公共图书馆

#### 5. 奢侈品产业链
```
原料采集 → 精加工 → 上层消费
```
- 升级: 贸易网络、奢侈品垄断

#### 6. 军事产业链
```
原料供应 → 武器制造 → 军队训练 → 军事行动
```
- 升级: 标准化武器、军事后勤、军工厂

### 产业链机制

#### 效率加成系统
- ✅ 完整产业链加成 (+20% 效率)
- ✅ 专业化加成 (+25% 效率)
- ✅ 规模经济 (最高 -30% 成本, +20% 效率)
- ✅ 技术外溢 (+15% 相关产业)

#### 瓶颈系统
- ⚠️ 资源短缺 (-30% 效率)
- ⚠️ 加工瓶颈 (-25% 效率)
- ⚠️ 劳动力短缺 (-40% 效率)
- ⚠️ 基础设施限制 (-20% 效率)

#### 发展路径
每条产业链都有多条发展路径可选择：
- 集约化 vs 粗放式
- 重工业 vs 精密制造
- 大众教育 vs 精英研究

## 🔗 四、系统联动机制

### 国家 ↔ 政令联动
- 不同国家对政令有增强或限制效果
- 例: 草原汗国的征兵法效率 +20%, 成本 -25%
- 例: 革命共和国的封建特权稳定度 -25

### 阶层 ↔ 政令反馈
- 阶层好感度影响政令效果
- 高好感 (>70%): 生产 +10%, 效率 +15%
- 低好感 (<30%): 生产 -15%, 效率 -20%

### 产业链 ↔ 阶层互动
- 产业链发展提升相关阶层好感度和财富
- 产业链衰退导致阶层不满和失业

### 产业链 ↔ 国家加成
- 草原汗国: 粮食链 +15%, 军事链 +25%
- 沙漠商队: 奢侈品链 +30%
- 丝绸帝国: 知识链 +25%, 奢侈品链 +20%

### 产业链 ↔ 政令效果
- 行会特许: 木材链、采矿链 +15%
- 自由贸易: 奢侈品链 +25%
- 基建计划: 所有产业链 +20%
- 战时经济: 军事链 +50%, 其他链 -20%

### 时代系统效果
- 不同时代解锁不同产业链
- 阶层结构随时代演变
- 政令效率随时代提升 (70% → 130%)

### 动态平衡机制
- ⚖️ 阶层平衡 (主导阶层惩罚, 弱势阶层保护)
- ⚖️ 经济平衡 (通货膨胀, 经济衰退)
- ⚖️ 军事平衡 (过度军事化惩罚)
- ⚖️ 文化平衡 (文化霸权加成)

### 随机事件系统
基于系统状态触发的动态事件：
- 📦 资源发现、技术突破、供应链中断
- 👥 阶层起义、黄金时代
- 🤝 贸易协定、外交危机

## 🚀 使用方法

### 1. 导入配置
```javascript
import {
  COUNTRIES,
  DECREES,
  INDUSTRY_CHAINS,
  NATION_DECREE_SYNERGIES,
  CHAIN_CLASS_INTERACTION
} from './config';
```

### 2. 选择国家
```javascript
const nation = COUNTRIES.find(c => c.id === 'silk_empire');
// 应用国家特性和能力
```

### 3. 激活政令
```javascript
const decree = DECREES.find(d => d.id === 'free_trade');
decree.active = true;
// 应用政令效果和检查组合
```

### 4. 发展产业链
```javascript
const chain = INDUSTRY_CHAINS.food_chain;
const efficiency = calculateChainEfficiency('food_chain');
// 升级产业链和选择发展路径
```

### 5. 应用系统联动
```javascript
// 检查国家对政令的影响
const synergy = NATION_DECREE_SYNERGIES[nation.id];
// 应用阶层反馈效果
// 触发随机事件
```
